## Description

This project showcases a personal portfolio including an "About Me" section, skills display, work/projects showcase, services offered, and contact information. It features a responsive design that works on both desktop and mobile devices.

## Features

- **Responsive Navigation**: Includes a hamburger menu for mobile devices.
- **Typewriter Effect**: A dynamic text effect in the hero section.
- **Smooth Scrolling**: Navigation links scroll smoothly to the corresponding section.
- **Active Link Highlighting**: Navigation links update based on the current scroll position.
- **Skill Bars Animation**: Animated progress bars for skills.
- **Project Showcase**: Links to live projects and GitHub repositories.

## Getting Started

To get a local copy of this project up and running, follow these simple steps.

### Prerequisites

*   A modern web browser (Chrome, Firefox, Safari, Edge).
*   No additional software is required to run the site locally.

### Installation

1.  **Clone the repository:**

    ```bash
    git clone https://github.com/tghimanshu/tghimanshu.github.io-old.git
    ```

    Or download the ZIP file and extract it.

2.  **Navigate to the project directory:**

    ```bash
    cd tghimanshu.github.io-old
    ```

## Usage

1.  Open the `index.html` file in your preferred web browser.
2.  Navigate through the sections using the top navigation bar.
3.  Scroll down to see various animations like the navbar background change and skill bars filling up.
4.  View the project on a mobile device (or resize the browser window) to see the responsive layout and hamburger menu.

## Technologies Used

*   **HTML5**: Markup structure.
*   **CSS3**: Styling and responsiveness.
*   **JavaScript (ES6+)**: Interactive logic.
*   **jQuery**: Used for DOM manipulation, event handling, and animations.
*   **Font Awesome**: Icon library.
